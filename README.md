# 🚀 Static Website Deployment with CI/CD using AWS S3

## 📌 Project Description

This project demonstrates how to deploy a static website using AWS S3 and automate the deployment process using GitHub Actions (CI/CD).

## 🛠️ Technologies Used

* AWS S3 (Static Website Hosting)
* AWS IAM (Access Management)
* GitHub (Version Control)
* GitHub Actions (CI/CD Pipeline)
* HTML (Frontend)

## ⚙️ Architecture

GitHub → GitHub Actions → AWS S3 → Live Website

## 🪜 Steps Involved

1. Created a static website using HTML
2. Hosted the website on AWS S3
3. Configured bucket policy for public access
4. Created IAM user and access keys
5. Stored AWS credentials in GitHub Secrets
6. Created a CI/CD pipeline using GitHub Actions
7. Automated deployment to S3 on every push

## 🌐 Output

The website is successfully hosted on AWS S3 and automatically updates whenever changes are pushed to GitHub.

## 🎯 Result

CI/CD pipeline is working successfully, enabling automatic deployment of website updates.
